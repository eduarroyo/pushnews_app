# Ti.VideoPlayer

### com.caffeinalab.titanium.videoplayer

## Installation

#### Via Gittio

```
gittio install com.caffeinalab.titanium.videoplayer
```

#### Via Github

Download the latest release, and add in your *config.json*, under `dependencies`:

```json
"dependencies": {
    "com.caffeinalab.titanium.videoplayer": "*"
}
```
